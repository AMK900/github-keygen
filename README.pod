=head1 NAME

github-keygen - bootstrap your Github SSH configuration

=head1 DESCRIPTION

The C<master> branch is now deprecated. Please switch to the C<release> branch
to run the latest release of C<github-keygen>.

    git checkout release

Or from scratch:

    git clone https://github.com/dolmen/github-keygen.git
    cd github-keygen
    perldoc github-keygen
